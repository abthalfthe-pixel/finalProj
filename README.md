# מערכת מבחנים מלאה בדפדפן (Client-Side Exam System)

פרויקט זה מיישם מערכת לניהול וביצוע מבחנים הפועלת בצד הלקוח בלבד, מבוססת על ארכיטקטורה נקייה, מונחית עצמים (OOP) וחלוקה מודולרית (ES Modules).

## 🚀 פיצ'רים מרכזיים במערכת

### 🔑 מערכת משתמשים והרשאות (Auth)
* **הרשמה והתחברות:** אפשרות ליצירת משתמש חדש ובחירת תפקיד במערכת (מורה או סטודנט).
* **ניתוב דינמי:** המערכת מזהה את סוג המשתמש ומעבירה אותו אוטומטית למסך המתאים לו.
* **אבטחת גישה:** הגנה על דפי הניהול (הדשבורדים) מפני כניסה לא מורשית ללא חיבור מתאים.

### 👩‍🏫 פאנל מורה (Teacher Dashboard)
* **יצירת מבחן חדש:** הגדרת שם למבחן והוספת שאלות ללא הגבלה.
* **ניהול שאלות אמריקאיות:** הזנת שאלה, 4 אפשרויות בחירה וקביעת התשובה הנכונה.
* **ניהול מבחנים:** צפייה ברשימת המבחנים הקיימים ואפשרות למחיקת מבחנים מהמערכת.

### 👨‍🎓 פאנל סטודנט (Student Dashboard)
* **חיפוש וסינון:** מנגנון חיפוש דינמי בזמן אמת לאיתור מבחנים לפי שמם.
* **ביצוע מבחן פעיל:** ממשק מענה מבוסס כפתורי רדיו (Radio Buttons) המאפשר בחירת תשובה אחת לכל שאלה.
* **הצגת תוצאות מיידית:** חישוב הציון והצגת אחוזי הצלחה (כולל חיווי על עובר/נכשל בציון 60 ומעלה) מיד עם שליחת המבחן.
* **היסטוריית ציונים:** שמירה מוחלטת של ההגשות והצגת ממוצע הציונים הכללי לצד טבלת היסטוריית מבחנים מפורטת.
